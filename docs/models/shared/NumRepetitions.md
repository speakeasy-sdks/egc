# NumRepetitions

the number of non-echo repeats of a broadcast (so total number of broadcasts not 
including echoes will be that number plus one being the initial broadcast)



## Values

| Name             | Value            |
| ---------------- | ---------------- |
| `ONCE`           | ONCE             |
| `TWICE`          | TWICE            |
| `TILL_CANCELLED` | TILL_CANCELLED   |
